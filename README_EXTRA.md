## 🔧 Configuration

### Environment Variables
```env
API_PORT=8000
DEBUG=True
DATA_PATH=./data/streams
PATHWAY_CACHE=./cache/pathway_storage
```

### Streamlit Settings
```toml
API_BASE = "http://localhost:8000"
THEME = "light"
```
